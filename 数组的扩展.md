# 数组的扩展
1. 类数组：**必须拥有length属性**，其它属性（索引）为非负整数(对象中的索引会被当做字符串来处理，这个不是必须的).如：函数参数arguments，nodelist,如下均为类数组。
```
let arrayLike = {
    "1":"a",
    "2":"b",
    length:2
 };

let arrayLike = {
    "b":"a",
    "a":"b",
    length:2
 };
```
2. Array.from
    - 用于将两类对象转为真正的数组：类数组的对象和可遍历（iterable）的对象（包括ES6新增的数据结构Set和Map及实现了Iterator接口的结构）。
    - 可以将字符串转换为数组，长度为字符串长度。
